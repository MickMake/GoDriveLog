# Latching and state-retention behaviour

**Gauge group:** [Dot-matrix or cell array](<../README.md>)

## Definition

A displayed state that remains mechanically, magnetically, electrically or optically retained until reset or rewritten.

This is a canonical umbrella label. The exact per-gauge wording and any qualifiers below remain the authoritative detail; gauges grouped here may reach the same visible symptom through different physics.
