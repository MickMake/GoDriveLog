# Scale direction and interpretation

**Gauge group:** [Radial pointer](<../README.md>)

## Definition

Whether increasing values move clockwise, anticlockwise, upward or otherwise, and how that direction is understood.

This is a canonical umbrella label. The exact per-gauge wording and any qualifiers below remain the authoritative detail; gauges grouped here may reach the same visible symptom through different physics.

### [X21 — Automotive econometer / manifold-vacuum gauge](<../gauges/x21_automotive_econometer_manifold_vacuum_gauge.md>)

- **Measured or indicated:** Engine manifold vacuum as a rough load/economy proxy
- **Era:** c.1930s-c.1980s common; aftermarket niche remains
- **Preserved source phrases:**
  - “High vacuum usually reads economical and low vacuum reads power, so scale meaning is inverse to throttle”
