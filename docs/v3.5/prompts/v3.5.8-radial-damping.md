# v3.5.8 Radial Damping

Implement the v3.5.8 slice only.

## Goal

Add lagged radial needle movement so radial gauges do not teleport to new values.

## Requirements

- Use the shared finite movement lifecycle.
- Keep movement bounded and deterministic.
- Default behaviour must remain current/immediate unless damping is configured.
- Add tests for configured damping, disabled/default behaviour, and final settled angle.
- Add a single-feature visual inspection fixture.

## Do not

- Do not add overshoot, stiction, peg bounce, hysteresis, or idle vibration in this slice.

## Completion

Update `docs/v3.5/ImplementationState.md` to mark v3.5.8 complete and set the next target to v3.5.9.
