# v3.5.7 Odometer Carry-Drag

Implement the v3.5.7 slice only.

## Goal

Add mechanical carry-drag so a higher digit begins to creep while the lower digit approaches rollover.

## Requirements

- Focus on rollover behaviour such as `9 -> 0`.
- Preserve existing `smooth` and `click` base modes.
- Keep behaviour deterministic and bounded.
- Add tests for disabled/default behaviour and enabled carry-drag.
- Add a single-feature visual inspection fixture with a small value range that makes rollover easy to inspect.

## Do not

- Do not add backlash, snap/settle, overshoot, or idle movement.
