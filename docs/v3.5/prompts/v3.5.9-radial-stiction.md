# v3.5.9 Radial Stiction

Implement the v3.5.9 slice only.

## Goal

Add sticky-needle threshold behaviour for radial gauges.

## Requirements

- Needle should hold for small changes below a configured threshold.
- Once threshold is exceeded, movement may release toward the target.
- Keep behaviour deterministic and display-only.
- Add validation for sensible thresholds.
- Add tests for below-threshold hold, above-threshold release, and disabled/default behaviour.
- Add a single-feature visual inspection fixture.

## Do not

- Do not add random stiction.
- Do not add overshoot, peg bounce, hysteresis, or idle vibration in this slice.
