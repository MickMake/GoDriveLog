## Goal

Add mechanical snap into final digit position with a small settle movement for odometers.

## Requirements

- Preserve existing odometer movement semantics.
- Do not change the accepted `odometer.movement` values or fallback behaviour in this slice.
- Treat snap/settle as a finite post-movement tail after the main odometer movement phase.
- Keep amplitude and timing configured, bounded, and deterministic.
- Add tests for enabled snap/settle, disabled/default behaviour, and final settled digit position.
- Add a single-feature visual inspection fixture, likely using a simple main movement mode if that gives the clearest visual result.

## Do not

- Do not add backlash in this slice.
- Do not add idle movement.
- Do not redefine the odometer main movement values in this slice.
