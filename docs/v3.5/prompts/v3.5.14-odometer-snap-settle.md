# v3.5.14 Odometer Snap / Settle

Implement the v3.5.14 slice only.

## Goal

Add mechanical snap into final digit position with a small settle movement for odometers.

## Requirements

- Preserve `movement: smooth` and `movement: click` as base modes.
- Treat snap/settle as finite movement after a value change.
- Keep amplitude and timing configured, bounded, and deterministic.
- Add tests for enabled snap/settle, disabled/default behaviour, and final settled digit position.
- Add a single-feature visual inspection fixture, likely click-focused if that gives the clearest visual result.

## Do not

- Do not add backlash in this slice.
- Do not add idle movement.

## Completion

Update `docs/v3.5/ImplementationState.md` to mark v3.5.14 complete and set the next target to v3.5.15.
