## Goal

Add an optional radial needle drop shadow that gives the rotating needle a small amount of visual depth.

This is a deterministic radial rendering feature. It is not an asset/layer system change and it is not dynamic parallax.

## Requirements

- Apply to `radial` gauges only.
- Preserve current radial rendering when the feature is absent or disabled.
- Add package/config support for an optional needle shadow setting.
- The shadow should render as an offset, tinted/alpha copy of the same rotating needle geometry.
- The shadow must use the same pivot, angle, scale, and value-to-angle result as the real needle.
- Render order should be: dial/background, needle shadow, real needle, existing foreground/overlay layers.
- Provide a simple configurable offset, for example `[x, y]` pixels.
- Provide a simple configurable alpha, clamped or validated to a sane range.
- Add tests for default-disabled behaviour, config parsing/validation, render model output, and unchanged real needle angle.
- Add a single-feature visual inspection fixture.

## Do not

- Do not add blur in this slice.
- Do not add dynamic light direction, gyro/view-angle parallax, or moving shadows.
- Do not add idle animation or ambient effects.
- Do not change existing asset/layer semantics.
- Do not apply this to non-radial gauge types.
- Do not alter source values or value mapping.
