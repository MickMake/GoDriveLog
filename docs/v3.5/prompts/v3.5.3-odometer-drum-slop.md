## Goal

Add fixed per-wheel odometer alignment offsets so drums can sit slightly imperfectly while static.

## Requirements

- Treat drum slop as static imperfection, not animation.
- Keep behaviour deterministic.
- Prefer explicit offsets over generated randomness.
- Preserve `movement: smooth` and `movement: click`.
- Add validation for sensible offset bounds.
- Add tests for configured offsets and default no-offset behaviour.
- Add a single-feature visual inspection fixture.

## Do not

- Do not add random runtime offsets.
- Do not add easing, carry-drag, backlash, snap/settle, or transition lifecycle in this slice.
