# v3.5.10 Radial Overshoot

Implement the v3.5.10 slice only.

## Goal

Add bounded radial needle overshoot for larger value changes.

## Requirements

- Overshoot must be configured, small, and bounded.
- Final settled value must match the target display value.
- Prevent misleading overshoot into danger/warning zones unless explicitly allowed.
- Add tests for disabled/default behaviour, bounded overshoot, and final settlement.
- Add a single-feature visual inspection fixture.

## Do not

- Do not add idle vibration, peg bounce, stiction changes, or hysteresis in this slice.
