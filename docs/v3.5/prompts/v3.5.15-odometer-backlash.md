# v3.5.15 Odometer Backlash

Implement the v3.5.15 slice only.

## Goal

Add direction-change slack/settle for worn odometer gear feel.

## Requirements

- Preserve `movement: smooth` and `movement: click` as base modes.
- Backlash should only respond to direction changes.
- Keep slack distance and settle timing configured, bounded, and deterministic.
- Add tests for forward movement, reverse movement, direction change, disabled/default behaviour, and final settled value.
- Add a single-feature visual inspection fixture.

## Do not

- Do not add random gear noise.
- Do not add idle movement.
- Do not modify source values.
