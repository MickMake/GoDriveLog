# v3.5.18 Radial Calibration Offset

Implement the v3.5.18 slice only.

## Goal

Add an optional display-only radial calibration offset so a needle can sit slightly imperfectly, like a real gauge that is not perfectly zeroed.

This affects only the final rendered needle angle.

## Requirements

- Apply to `radial` gauges only.
- Preserve current radial value-to-angle behaviour when the feature is absent or set to zero.
- Add package/config support for an optional calibration offset in degrees.
- Apply the offset after the normal value-to-angle calculation and before rendering.
- Keep the offset display-only.
- Respect existing min/max angle limits or configured clamping behaviour.
- Add tests for default-zero behaviour, positive offset, negative offset, clamping/bounds, and unchanged input value.
- Add a single-feature visual inspection fixture.

## Do not

- Do not change input values.
- Do not change exported values or configured value ranges.
- Do not add random drift.
- Do not add idle animation.
- Do not apply this to non-radial gauge types.
- Do not combine this slice with dynamic parallax, lighting, or shadow movement.
