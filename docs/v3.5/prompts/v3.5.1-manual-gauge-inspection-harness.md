# v3.5.1 Manual Gauge Inspection Harness

Implement the v3.5.1 slice only.

## Goal

Add an interactive visual inspection mode for one gauge at a time.

## Required behaviour

- Keep the existing sweep/heartbeat harness behaviour.
- Add a new inspection mode or pattern beside the existing harness options.
- Load one normal YAML gauge/dashboard fixture.
- Display one gauge.
- Start at the midpoint of the configured value range.
- Left arrow jumps to min.
- Right arrow jumps to max.
- Up arrow increments.
- Down arrow decrements.
- Shift + Up/Down uses a coarse step.
- Ctrl/Cmd + Up/Down uses a fine step.
- R resets to midpoint.
- Space replays the last transition where possible.
- Esc/Q quits.
- Mouse wheel may increment/decrement if simple to support.

## Fixture rules

- Use normal YAML fixtures; do not add a special metadata layer.
- Add baseline and single-feature fixture structure if useful.
- Permit one deliberate `99-all-options` fixture per gauge type.

## Do not

- Do not add screenshots, videos, or visual diff reports.
- Do not implement any realism feature in this slice.
