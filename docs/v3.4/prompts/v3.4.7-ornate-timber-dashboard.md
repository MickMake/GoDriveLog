## Theme name

Use:

```text
ornate-timber
```

## Visual direction

The dashboard should say:

```text
Oh. So you're a master carpenter, eh?
```

Use multiple timber treatments and strong contrast:

- carved or routed timber panel background;
- contrasting timber species/tints such as dark walnut-like, red jarrah-like, pale maple-like, and oak-like tones;
- timber needles;
- timber tick marks or inlaid tick details;
- routed timber bezels or rings;
- dark recessed gauge faces so the timber details remain readable;
- subtle screw heads, plugs, joinery lines, or decorative inlays;
- amber or green numeric displays behind smoked acrylic where useful.

Avoid brown soup. The dashboard needs readable instruments, not a very expensive chopping board with telemetry.

## Scope

Add only the ornate timber themed dashboard and its generated assets.

Expected output may include:

```text
examples/assets/v3.4/ornate-timber/
examples/dashboards/ornate-timber.yaml
```

Use the generation framework from v3.4.6. Extend the framework only where needed for this theme.

## Gauge coverage target

Cover as much of the completed v3.4 gauge model as practical:

- `radial` RPM or speed gauge with timber needle and timber/inlay ticks;
- `numeric` display with amber/green asset treatment;
- `odometer` with a timber/framed mechanical look;
- `indicator` warning lamps or engraved inlay indicators;
- `bar` fuel/temperature level in a carved slot;
- `segmented` stepped indicator or level display using timber/inlay artwork.

Do not create new gauge behaviour to achieve the look.

## Asset rules

- Generate all assets deterministically.
- Keep transparent layers where useful.
- Use seeded procedural timber grain/noise.
- Keep generated source/config committed.
- Do not use remote image generation.
- Do not download stock timber textures.
- Do not hand-edit generated PNGs as the source of truth.

## Runtime rules

- Use existing v3.4 gauge types only.
- Do not add runtime `style` fields.
- Do not alter renderer behaviour unless fixing a genuine example-loading bug.
- Keep decorative joinery, screw heads, plugs, and inlays as image assets.
