## Goal

Render the average pointer marker for radial and bar gauges using the shared average marker engine.

## Asset contract

Radial gauges use:

```text
needle_average.png
```

Bar gauges use:

```text
marker_average.png
```

## Render order

Radial:

```text
background -> face -> ticks -> live needle -> pointer markers -> overlay/glass/bezel
```

Bar:

```text
background/base -> bar track/face -> live bar/fill/indicator -> pointer markers -> overlay/frame/glass
```

Average markers render above the live indicator and below the final overlay/glass/bezel/frame, the same as min/max markers.

## Requirements

- Preserve all prior v3.6 marker behaviour.
- Apply to radial and bar gauges in this slice.
- Preserve current rendering when `average` is absent or disabled.
- Render radial average marker with `needle_average.png` where available.
- Render bar average marker with `marker_average.png` where available.
- Use the same radial pivot/rotation model as the live radial needle.
- Respect existing bar horizontal, vertical, reversed, and origin/direction configuration.
- Keep average marker assets visually distinct from the live indicator and min/max marker assets.
- Add tests or render model checks for radial average rendering, bar average rendering, disabled behaviour, layer order, and asset selection.
- Add visual inspection fixtures for radial and bar gauges showing min, max, and average together.

## Do not

- Do not calculate a statistical average.
- Do not replace explicit marker PNG assets with arbitrary procedural shapes.
- Do not alter min/max marker behaviour.
- Do not add persistence.
- Do not add a `source` switch.
