## Goal

Add an optional bar damped secondary pointer marker.

This marker is a slow secondary bar indicator inspired by mechanically damped gauges. It is not a mathematical average and must not be documented or tested as an arithmetic mean.

## Config

Preferred config shape:

```yaml
realism:
  pointer_markers:
    damped:
      enabled: true
```

The config key is shared with radial gauges, but rendering is gauge-family specific.

## Requirements

- Apply to `bar` gauges only in this slice.
- Preserve bar min/max marker behaviour.
- Preserve radial marker behaviour.
- Preserve current bar rendering when `realism.pointer_markers.damped` is absent or disabled.
- Add package/config support for bar `realism.pointer_markers.damped`.
- Render a secondary bar marker that moves more slowly/smoothly than the live bar fill/indicator.
- Respect existing bar orientation and direction rules.
- Use a deterministic damping model compatible with existing bar movement code where possible.
- Keep the marker display-only; do not mutate source values, logs, exports, configured ranges, or input data.

- Add a single-feature visual inspection fixture.
