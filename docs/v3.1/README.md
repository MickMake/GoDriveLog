# GoDriveLog v3.1

Planning documents for the v3.1 implementation series.

PR 51 creates this planning baseline. It is docs-only and is not a numbered implementation slice.

## Documents

- `ReleasePlan.md` - implementation roadmap and version queue.
- `MigrationState.md` - current v3.1 state and next target.
- `OpenDecisions.md` - unresolved decisions with blocking and impact notes.
- `CarryForward.md` - v3.0 and v2 lessons that still matter.
- `prompts/README.md` - common implementation workflow, guardrails, and review checklist.
- `prompts/v3.1.x-*.md` - one prompt per implementation slice.

## First implementation slice

Implementation starts at `v3.1.0-runnable-command-path`.

Planning setup is deliberately outside the version queue. Tiny app, fewer steering wheels.
