## Scope

Extend or configure the existing v3 dashboard harness so a gauge widget can be exercised through the real runtime path.

## Behaviour

- Use fake sensor events through the existing v3 dashboard runtime path.
- Exercise a fixed gauge value.
- Exercise a sweep pattern across the gauge value map.
- Exercise non-`ok` states.
- Confirm the gauge scene/render path does not bypass runtime boundaries.

## Out of scope

- New UI controls unless truly necessary.
- Major harness redesign.
- Dashboard event efficiency optimisation.
- Retirement readiness.
- Clusters.
- Gauge inheritance.

## Acceptance checks

- Harness can show or exercise a gauge widget.
- Fixed value works.
- Sweep values work.
- Non-`ok` states do not render fake live gauge values.
- Existing harness behaviours still work.
- `CHANGES.md` is updated.
- `docs/v3.2/ImplementationState.md` is updated.
