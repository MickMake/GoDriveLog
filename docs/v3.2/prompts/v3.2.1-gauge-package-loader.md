## Scope

Add the minimal data model and loader support for gauge packages.

A gauge package is any directory under `assets/gauges/` containing a file named `gauge.yaml`.

Directory names under `assets/gauges/` carry no semantic meaning.

## Behaviour

The loader should:

- accept a dashboard gauge path such as `assets/gauges/7Seg/amber/4_digit_rpm`;
- resolve it to `assets/gauges/7Seg/amber/4_digit_rpm/gauge.yaml`;
- parse the gauge YAML;
- support at least `type: seven_segment` and `type: radial` in parsed packages;
- preserve sensor binding from `sensor`;
- preserve package-owned fields such as `format`, `digit_set`, `digits`, `layers`, `pivot`, and `value_map` where applicable;
- resolve layer image paths relative to the `gauge.yaml` directory;
- allow relative paths like `../7Seg0.png` or `../../7Seg4Digits.png` when they remain inside the asset tree;
- reject path traversal outside the asset tree;
- return clear validation/load errors.

## Minimal seven-segment gauge.yaml shape

```yaml
id: amber_4_digit_rpm
type: seven_segment
sensor: rpm
format: "%04.0f"

size:
  width: 398
  height: 150

layers:
  panel: ../../7Seg4Digits.png
  glass: ../../Glass.png

digit_set:
  background: ../../7SegBack.png
  characters:
    "0": ../7Seg0.png
    "1": ../7Seg1.png
    "2": ../7Seg2.png
    "3": ../7Seg3.png
    "4": ../7Seg4.png
    "5": ../7Seg5.png
    "6": ../7Seg6.png
    "7": ../7Seg7.png
    "8": ../7Seg8.png
    "9": ../7Seg9.png
    "-": ../7SegMinus.png
  decimal_point: ../7SegDP.png
  spacing: 4

digits:
  count: 4
  positions:
    - [35, 35]
    - [117, 35]
    - [199, 35]
    - [281, 35]
```

## Minimal radial gauge.yaml shape

```yaml
id: simple_radial_rpm
type: radial
sensor: rpm

size:
  width: 512
  height: 512

layers:
  background: ../shared/radial/simple_rpm/bezel.png
  face: ../shared/radial/simple_rpm/face.png
  ticks: ../shared/radial/simple_rpm/ticks.png
  needle: ../shared/radial/simple_rpm/needle.png
  overlay: ../shared/radial/simple_rpm/glass.png

pivot:
  face: { x: 0.5, y: 0.55 }
  needle: { x: 0.5, y: 0.9 }

value_map:
  min: 0
  max: 7000
  start_angle: -135
  end_angle: 135
  clamp: true
```
