Use a branch named:

```text
v3.2.4-fyne-seven-segment-rendering
```

Branch from latest `main` after confirming the v3.2.3 PR is merged.

## Scope

Add Fyne adapter support for complete seven-segment gauge packages.

## Behaviour

- Render static display layers such as panel/bezel and glass.
- Render digit glyphs using the package-configured digit set.
- Place digits using package-owned digit positions.
- Scale the complete package using the widget scale.
- Place the complete package using the widget position.
- Preserve existing rendering for current widget types.
- Keep Fyne-specific image handling inside the display adapter.

## Out of scope

- Gauge package loading changes.
- Dashboard config changes.
- Radial gauge rendering.
- Procedural drawing.
- Animation.
- Sensor overrides.
- Gauge inheritance.
- Clusters.

## Acceptance checks

- A seven-segment gauge scene renders static panel/bezel and glass layers.
- A seven-segment gauge scene renders digit glyphs in configured positions.
- Widget position and scale apply to the complete mounted display.
- Non-`ok` states do not render fake live digit values.
- Existing widget rendering still works.
- `CHANGES.md` is updated.
- `docs/v3.2/ImplementationState.md` is updated.
